Основные команды и действия

Начало работы

Устанавливаем gulp: npm install gulp

Проверяем текущие установленные версии:

node --version (-v)

npm -v

npx -v

После создания папки под проект, создаем файл .json

npm init (-y)

Устанавливаем пакет gulp в наш проект

npm install --save-dev gulp

Проверяем текущую версию gulp: gulp --version

Создаем файл gulpfile.js

!Важно. "devDependencies" и "dependencies"

"devDependencies" - для плагинов и разработки0 то что не попадет в финальную версию (префиксы, минификация и тд)

"dependencies" - подключение скриптов (bootstrap, typeScript, Jquary и тд), то что попадет в финальную версию проекта
